Sobre o Programa:

Este programa foi feito para a conclusão do primeiro semestre do curso de Desenvolvimento de Sistemas, na escola Senai Prof. DR. Euryclides de Jesus Zerbini. Criado por quatro alunos para uma Pet Shop que precisava de um programa que facilitasse o cadastro de clientes e o agendamento de serviços como banho, tosa, tosa higiênica, etc. Abaixo, nos cards a seguir você pode conferir todas as funcionalidades que o programa apresenta, além de uma facilidade extrema para utiliza-lo:

Cadastro

Aqui começa o programa, na tela onde será cadastrado no sistema o cliente, apenas preenchendo os dados e clicando em cadastrar é o suficiente.

Serviços

Para uma melhor organização, colocamos separado o cadastro dos serviços, e aqui segue o mesmo esquema do cadastro, basta apenas colocar os tipos de serviço desejado e salvar.

Pesquisar

Na parte da pesquisa, você poderá consultar os horários agendados para o dia desejado, qual cliente está marcado e o tipo de serviço.

Salvar

Para garantir que está tudo salvo, ao sair do programa, aparecerá uma mensagem perguntando se desejar salvar os horários agendados ou descarta-los
